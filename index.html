سياسة الخصوصية

تطبيق محامي الجزائر - مساعدك القانوني الشخصي

آخر تحديث: 8 مايو 2025

نحن في تطبيق "محامي الجزائر" نقدر خصوصيتك ونلتزم بحماية بياناتك الشخصية. تشرح سياسة الخصوصية هذه كيفية جمع واستخدام وحماية المعلومات التي تقدمها عند استخدام تطبيقنا.

١. المعلومات التي نجمعها

نقوم بجمع المعلومات التالية عند استخدامك للتطبيق:

  • محتوى المحادثات: النصوص والاستفسارات التي تقوم بإدخالها في التطبيق للحصول على المعلومات القانونية.
  • الصور المرفوعة: أي صور أو وثائق تقوم بتحميلها للتحليل القانوني.
  • مفتاح API: في حالة استخدامك لمفتاح API خاص بك للاتصال بخدمات OpenAI.

٢. كيفية استخدام المعلومات

نستخدم المعلومات التي نجمعها للأغراض التالية:

  • تقديم خدمات المساعدة القانونية المطلوبة.
  • تحسين وتطوير خدمات التطبيق.
  • تخزين محفوظات المحادثات الخاصة بك لتتمكن من الرجوع إليها لاحقًا.
  • معالجة الصور والوثائق التي تقوم بتحميلها للحصول على تحليل قانوني.

٣. تخزين البيانات

تتم معالجة وتخزين بياناتك كما يلي:

  • التخزين المحلي: يتم تخزين محفوظات المحادثات محليًا على جهازك باستخدام UserDefaults، ولا يتم مشاركتها مع خوادمنا.
  • مفتاح API: يتم تخزين مفتاح API الخاص بك محليًا على جهازك فقط ولا يتم مشاركته مع أي طرف ثالث باستثناء OpenAI لتنفيذ الطلبات.
  • معالجة الصور: عند تحميل صورة للتحليل، يتم إرسالها إلى OpenAI عبر API آمن. لا نقوم بتخزين الصور على خوادمنا.

٤. مشاركة البيانات مع الأطراف الثالثة

نشارك بياناتك فقط مع الجهات التالية:

  • OpenAI: نستخدم خدمات OpenAI لمعالجة استفساراتك وتحليل الصور. تخضع البيانات المرسلة إلى OpenAI لسياسة الخصوصية الخاصة بهم.
  • لا نبيع أو نؤجر معلوماتك الشخصية لأطراف ثالثة لأغراض تسويقية.

ملاحظة مهمة: تطبيقنا يستخدم واجهة برمجة تطبيقات (API) من OpenAI لتوفير الإجابات على الاستفسارات القانونية وتحليل الصور. عندما تستخدم هذه الميزات، قد يتم إرسال نصوصك وصورك إلى خوادم OpenAI للمعالجة. نوصي بعدم مشاركة معلومات سرية للغاية أو بيانات شخصية حساسة.

٥. أمان البيانات

نحن نتخذ تدابير أمنية معقولة لحماية معلوماتك الشخصية من الوصول غير المصرح به أو الإفصاح. ومع ذلك، لا يمكن ضمان أمان البيانات المرسلة عبر الإنترنت بشكل مطلق.

٦. حقوقك

لديك الحقوق التالية فيما يتعلق ببياناتك:

  • الوصول إلى بياناتك الشخصية المخزنة في التطبيق.
  • مسح محفوظات المحادثات من خلال ميزة "مسح المحادثات" المتوفرة في التطبيق.
  • حذف مفتاح API الخاص بك من التطبيق في أي وقت.

٧. إخلاء المسؤولية

المعلومات القانونية المقدمة من خلال التطبيق هي معلومات عامة ولا تعتبر استشارة قانونية رسمية. نوصي دائمًا بالرجوع إلى محام مختص للحصول على المشورة القانونية المتعلقة بحالتك الخاصة.
